Panda Factory

Getting started:
1) Get the virtual envornment running
    env\Scripts\activate
To stop using virtualenv, type "deactivate"
2) Start the server
    InternNest\manager.py runserver
3) In a web browser, visit:
    http://127.0.0.1:8000/

Tech Stack:
- Django 1.5.1
